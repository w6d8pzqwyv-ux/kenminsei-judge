# 県民性ジャッジ スコア設計書(v1・2026-07-09)

5軸(行動力・社交性・きちょうめん度・人情深さ・好奇心)のスコアを、出典つきデータから機械的に計算するルール。

## 基本方式:偏差値の合成

1. 各軸に2〜4個の「材料指標」を割り当てる(下表)
2. 指標ごとに47都道府県で**偏差値化**(平均50・標準偏差10。学力テストと同じ仕組み)
3. 軸のスコア = 割り当てた指標の偏差値の**平均**(初期は均等ウェイト)
4. 表示は数値そのままでなく**5段階のことば**に丸める(過度な精密さを装わない)
   - 62以上「とても高め」/ 55〜61「高め」/ 46〜54「ふつう」/ 39〜45「マイペース型」/ 38以下「じっくり型」
   - ※低い側も否定語を使わない(絶対禁止事項に準拠)

## 5軸×材料指標の対応表

| 軸 | 指標(すべて都道府県別に取得可) | 出典 |
|---|---|---|
| **行動力** | ①スポーツ行動者率 ②旅行・行楽行動者率 ③開業率 | 社会生活基本調査(e-Stat) / 中小企業白書 |
| **社交性** | ①交際費支出 ②外食費支出 ③お笑い芸人輩出数(人口比) | 家計調査 / とどラン |
| **きちょうめん度** | ①貯蓄率(貯蓄額÷収入) ②持ち家率 ③学力テスト正答率 | 全国家計構造調査 / 住宅・土地統計 / 文科省 |
| **人情深さ** | ①ボランティア行動者率 ②献血率 ③三世代同居率 | 社会生活基本調査 / 厚労省 / 国勢調査 |
| **好奇心** | ①書籍・教養娯楽支出 ②図書館貸出冊数(人口比) ③大学進学率 | 家計調査 / 日本図書館協会 / 学校基本調査 |

## 学術データによる補正(吉野・小塩2021)

- 査読論文が示す地方ブロックの傾向(例: 沖縄=神経症傾向 全国最低、首都圏=外向性高)と、上の統計合成値が**矛盾する場合は解説文で両論併記**する(数値はいじらない。透明性優先)
- 論文の47県数値が入手できたら(vol.3公開 or 手起こし)、統計合成と**50:50でブレンド**する将来オプションを残す

## 出典表示のルール

- 結果画面の下部に「この診断は◯◯(指標名)など計N個の公的統計・調査から算出しています」+ 出典リンク
- 各統計の**調査年を明記**(古いデータ混在のため)
- 家計調査系は「県庁所在市の調査」であることを注記

## 計算の実務(Phase 1)

- 47県ぶんの指標データはCSVで1回だけ手集め(またはe-Stat APIバッチ)→ JSON化してアプリに同梱
- 計算はビルド時に済ませ、アプリは完成品のJSONを読むだけ(ブラウザで毎回計算しない)

## 確定事項(2026-07-09 ユーザー決定)

1. 5段階のことば = 上記の案で確定(とても高め/高め/ふつう/マイペース型/じっくり型)
2. 「お笑い芸人輩出数」= 社交性に**入れる**(出典表示で「民間集計」と明記すること)
3. 郷土愛(ブランド総研・民間調査)= 人情深さに**入れない**(公的統計に絞る。解説文の"うんちく"としての引用は可)
