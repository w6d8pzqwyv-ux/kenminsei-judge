## 見送った候補(代表例)

| 候補 | 見送り理由 |
|---|---|
| 自治会・町内会加入率 | 総務省の全国横断調査が市区町村単位の個別集計中心で、47都道府県そろえた公的統計の一覧が見当たらなかった。民間サイトの独自集計はあるが出典の信頼性が弱い |
| インターネット利用率・SNS利用率 | 好奇心/社交性どちらにも使えそうだが、通信利用動向調査は都道府県別の公開粒度が粗く(ブロック単位が中心)、47都道府県が全部揃うか未確認のため保留 |
| 郷土愛(ブランド総研調査) | スコア設計.mdで既に「人情深さには入れない」と確定済み(民間調査・公的統計方針に反する) |
| 犯罪率・離婚率・自殺率など | AGENTS.mdの絶対禁止事項(ネガティブ属性を使わない)に抵触するため対象外 |
| 老人ホーム数・生活保護関連(J福祉・社会保障の大半) | 「福祉施設の量」であり個人の行動・性格特性との対応が弱い(高齢化率など別要因に引っ張られる)。人情深さとの結びつきが説明しにくいため見送り |

## 統括への申し送り(判断が必要な点)

1. **海外旅行行動者率・NPO法人数は2軸にまたがる候補**。海外旅行率は「行動力」寄りか「好奇心」寄りか、NPO法人数は「社交性」寄りか「人情深さ」寄りか、どちらか一方に採用するか両方に薄く使うか方針決定が必要(スコア設計.mdの「1指標1軸」原則を踏襲するなら片方選択を推奨)。
2. **きちょうめん度は追加候補が見つからなかった**。社会・人口統計体系のE(教育)・G(文化・スポーツ)を確認したが「計画性・几帳面さ」に直結する統計が乏しい。案として、家計調査系の「へそくり・保険加入率」や「公共料金の口座振替率」等、別カテゴリ(家計・行政基盤)を追加調査するか、無理に5指標にせず3〜4指標のままにするか判断が必要。
3. **NPO法人数(内閣府)・民生委員数(e-Stat)は「人数・施設数」系指標**であり、既存の「行動者率」系(個人の行動を直接聞いた調査)とは統計の性質が異なる(供給側 vs 需要側)。指標構成のバランスとして、性質の違う指標を混ぜてよいか確認したい。
4. **献血率(2009年)・図書館貸出冊数(2014年度)は古い**。今回追加候補(海外旅行率2021年・旅券発行2024年・転入率2024年)で置き換えor補強する余地がある。優先して新しい指標に差し替えるか、両論併記のまま残すか判断が必要。
5. 民生委員数(#J05101)・公民館数(#G01101)・ボランティア行動者率(#G041011)は今回タイトル一覧の確認どまりで、実データ(47都道府県ぶんの値)は未取得。採用が決まれば追加でAPI叩いてCSV化する。
